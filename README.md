This is a simple calculator app built using React.js. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
•	Addition
•	Subtraction
•	Multiplication
•	Division
•	Clear Button
•	Error Handling for Division by Zero

## Technologies Used
•	React.js
•	JavaScript
•	HTML
•	CSS
